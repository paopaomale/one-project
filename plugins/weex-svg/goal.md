#### weex-svg计划

预计开发时间`2周`左右。

一期计划，实现对常用元素的渲染实现，从而满足对于weex项目最基本的svg需求

``` html
circle,
ellipse,
g,
linear-gradient,
radial-gradient,
line,
path,
polygon,
polyline,
rect,
symbol,
text,
use,
defs,
stop
```

支持对部分表现属性的支持：
``` html
fill: 颜色填充
stroke： 描边
strokeWidth： 描边宽度
strokeLinecap: 线条头部形状
rotate： 旋转
x: x轴上的平移
y: y轴上的平移
```
